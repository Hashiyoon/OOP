# OOP
team oop project
Веб-сервис для автоматического формирования коммерческих предложений (КП) из базы данных индивидуального предпринимателя или малого бизнеса.

Основные функции
- Личный кабинет ИП: управление услугами, логотипом, печатью и шаблонами КП
- Личный кабинет покупателя: выбор услуг, формирование КП
- Генерация коммерческого предложения в формате PDF и Excel
- Проверка минимальных условий (количество, сумма, валидность)
- Роли пользователей: ИП и покупатель
- Поддержка монетизации (базовый и премиум-планы)

Технологии

**Backend:** JDK 25, JPA, PostgreSQL, Apache POI,   
**Frontend:** HTML, Swing  
**Auth:** JWT  
**Build:** Maven  
**Docs:** Swagger / OpenAPI
